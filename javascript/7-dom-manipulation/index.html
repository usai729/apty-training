DOM Manipulation

Topics

What is DOM?
The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree, where each node is an object representing a part of the document. JavaScript interacts with the DOM to manipulate HTML and CSS dynamically.
Accessing DOM Elements
JavaScript provides various methods to access DOM elements:
  • document.getElementById(): Selects an element by its ID.
  • document.getElementsByClassName(): Selects elements by their class name.
  • document.getElementsByTagName(): Selects elements by their tag name.
  • document.getElementsByName(): Selects elements by their name attribute.
  • document.querySelector(): Selects the first element matching a CSS selector.
  • document.querySelectorAll(): Selects all elements matching a CSS selector.
  • window.document: to access the root of the document and start traversal.
  • iframe.contentWindow: Refers to the window object of the iframe.
  • iframe.contentDocument: Refers to the document object inside the iframe.
  • element.shadowRoot: If the Shadow DOM is created with { mode: 'open' }, you can access it using the shadowRoot property
DOM Traversal
DOM traversal methods allow navigating between nodes:
  • element.parentNode: Access the parent of a node.
  • element.childNodes: Access all child nodes of a node.
  • element.firstChild and element.lastChild: Access the first and last child nodes.
  • element.nextSibling and element.previousSibling: Access the next and previous sibling nodes.
  • element.children: Access all child elements (ignoring text and comment nodes).
Manipulating DOM Elements
You can manipulate DOM elements using JavaScript:
  • element.innerHTML = "<p>New Content</p>";: Get or set the HTML content of an element.
  • element.textContent = "New Text Content";: Get or set the text content of an element.
  • element.setAttribute("class", "new-class");: Set an attribute on an element.
  • element.getAttribute("id");: Get an attribute on an element.
  • element.removeAttribute("disabled");: Remove an attribute on an element.
  • element.cloneNode(true);: Clone an element (with or without its children)
  • element.getBoundingClientRect(): Get position and dimensions of an element
  • element.dataset["data-key-name"]; element.dataset["data-key-name"] = 'value': Accessing and Updating Custom Data Attributes
  • element.scrollIntoView({ behavior: "smooth" });: Scroll to an element
  • element.remove();: Directly remove an element
  • element.replaceWith(newElement);: Replace an element with another
  • element.classList.add("active");
    element.classList.remove("hidden");
    element.classList.toggle("expanded");    : Add, remove, or toggle classes on an element.
  • element.style.color = "red";
    element.style.backgroundColor = "blue";    : Apply inline CSS styles dynamically.
  • inputElement.value = "New Value";: Change the value of an input field
  • checkboxElement.checked = true;: Check or uncheck a checkbox
  • window.scrollTo(0, 500);: Set or get the scroll position
Creating and Removing Elements
JavaScript allows you to create and remove DOM elements:
  • document.createElement('tagName');: Create a new element
  • parentElement.appendChild(newElement);: Add a new child element to a parent
  • parentElement.removeChild(childElement);: Remove a child element
  • parentElement.insertBefore(newElement, referenceElement);: Add a new child element before a specific element
  • parentElement.replaceChild(newElement, oldElement);: Replace one element with another element
