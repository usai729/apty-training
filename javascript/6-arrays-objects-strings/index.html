Arrays, Objects, and Strings

Topics

Arrays

1. What is an Array?
Arrays are used to store multiple values in a single variable. They are list-like objects in JavaScript that allow storing and manipulating collections of data.
2. Creating Arrays
  • Using square brackets: const arr = [1, 2, 3];
  • Using Array constructor: const arr = new Array(3);
3. Accessing and Modifying Elements
  • Accessing: Use the index: arr[0]
  • Modifying: Assign a new value: arr[1] = 42;
4. Common Array Methods
  • push: Adds elements to the end of an array.
  • pop: Removes the last element.
  • shift: Removes the first element.
  • unshift: Adds elements to the beginning.
  • slice: Returns a shallow copy of a portion of the array.
  • splice: Adds/removes elements in the array.
  • concat: Merges two or more arrays.
  • indexOf: Finds the first index of a value.
  • lastIndexOf: Finds the last index of a value.
  • join: Joins array of values and return string.
  • includes: Checks if a value exists in the array.
  • flat: Flattens nested arrays.
5. Iterating Over Arrays
  • for loop: Iterates using index.
  • for...of: Iterates directly over values.
6. Higher-Order Array Methods
  • forEach: Executes a callback for each element.
  • map: Creates a new array by applying a function to each element.
  • filter: Filters elements based on a condition.
  • reduce: Reduces the array to a single value.
  • find: Finds the first element matching a condition.
  • every: Checks if all elements match a condition.
  • some: Checks if any element matches a condition.
  • sort: Sorts the array.
  • reverse: Reverses the array order.
7. Multidimensional Arrays
Arrays within arrays. Example: const matrix = [[1, 2], [3, 4]];
8. Array Destructuring
Extract values from arrays into variables: const [a, b] = arr;
9. Spread and Rest Operators
  • Spread: Expands an array: [...arr]
  • Rest: Collects remaining elements: const [a, ...rest] = arr;
10. Checking Array Properties
  • isArray: Checks if a value is an array.
  • length: Gets the array length.
11. Working with Sparse Arrays
Sparse arrays have empty slots. Example: const arr = [1, , 3];
12. Array-Like Objects
Convert array-like objects like arguments to arrays with Array.from.

Objects

In JavaScript, objects are a collection of key-value pairs, where each key (also called a property) is a string, and its value can be any valid data type such as numbers, strings, arrays, functions, or even other objects. Objects are fundamental to JavaScript and are used to store and organize data in a structured way.

1. Creating an Object

You can create an object in two primary ways:

  • Object Literal Notation: This is the most common and easiest way to create an object.
    • const person = {
    name: 'John',
    age: 30,
    greet: function() {
        console.log('Hello ' + this.name);
    }
};
  • Using the new Object() Syntax: This is an alternative way of creating an object but is less common.
    • const person = new Object();
person.name = 'John';
person.age = 30;
person.greet = function() {
    console.log('Hello ' + this.name);
};
2. Accessing Object Properties

You can access object properties in two ways:

  • Dot Notation: This is the most common and easiest way to access properties. 
    • console.log(person.name); // 'John'
  • Bracket Notation: This is used when property names contain spaces, special characters, or when the property name is dynamically determined.
    • console.log(person['age']); // 30
3. Modifying Object Properties

You can modify properties of an object using dot notation or bracket notation.

  • Using Dot Notation:
    • person.name = 'Jane';
console.log(person.name); // 'Jane'
  • Using Bracket Notation:
    • person['age'] = 35;
console.log(person['age']); // 35
4. Object Methods

Objects can contain methods (functions) as their properties. You can call these methods just like any other function, but they are called on the object itself.

  • Adding a Method:
    • const person = {
    name: 'John',
    age: 30,
    greet: function() {
        console.log('Hello ' + this.name);
    }
};
  • Calling a Method:
    • person.greet(); // 'Hello John'

    Note: The this keyword inside the method refers to the object itself.

5. Object Destructuring

Object destructuring is a shorthand syntax to extract values from an object into variables. 

const person = {
    name: 'John',
    age: 30,
    job: 'Developer'
};

const { name, age } = person;
console.log(name); // 'John'
console.log(age);  // 30

You can also rename variables during destructuring:

const { name: fullName, age: yearsOld } = person;
console.log(fullName); // 'John'
console.log(yearsOld); // 30
6. Object Constructor

JavaScript provides a built-in Object() constructor to create objects.

const person = new Object();
person.name = 'John';
person.age = 30;

Alternatively, you can create a custom constructor function to instantiate objects.

function Person(name, age) {
    this.name = name;
    this.age = age;
}

const john = new Person('John', 30);
console.log(john.name); // 'John'
console.log(john.age);  // 30
7. Object.keys(), Object.values(), Object.entries()

JavaScript provides three methods to interact with the keys and values of an object:

  • Object.keys(): Returns an array of the object's keys.
    • const person = { name: 'John', age: 30 };
console.log(Object.keys(person)); // ['name', 'age']
  • Object.values(): Returns an array of the object's values. 
    • console.log(Object.values(person)); // ['John', 30]
  • Object.entries(): Returns an array of key-value pairs. 
    • console.log(Object.entries(person)); // [['name', 'John'], ['age', 30]]
8. Object.freeze() and Object.seal()
  • Object.freeze(): Makes an object immutable, meaning its properties cannot be added, deleted, or modified.
    • const person = { name: 'John', age: 30 };
Object.freeze(person);
person.age = 35; // This won't work
console.log(person.age); // 30
  • Object.seal(): Prevents new properties from being added, but existing properties can still be modified.
    • const person = { name: 'John', age: 30 };
Object.seal(person);
person.name = 'Jane'; // Works fine
delete person.age;    // Doesn't work
console.log(person);  // { name: 'Jane', age: 30 }
9. this Keyword in Objects

The this keyword refers to the current object in which a method is called. It can be used to access or modify the properties of the object.

const person = {
    name: 'John',
    greet: function() {
        console.log('Hello, ' + this.name); // `this` refers to the person object
    }
};

person.greet(); // 'Hello, John'
10. Prototypes and Inheritance

JavaScript objects can inherit properties and methods from other objects using prototypes. Every JavaScript object has a prototype, which is an object itself, from which the current object can inherit properties.

function Person(name, age) {
  this.name = name;
  this.age = age;
}

Person.prototype.greet = function() {
  console.log('Hello ' + this.name);
};

const john = new Person('John', 30);
john.greet(); // 'Hello John'

Strings

A string in JavaScript is a sequence of characters used for storing and manipulating text. Strings are one of the most commonly used data types in JavaScript, and they are immutable, meaning once a string is created, it cannot be changed directly.

1. String Creation

Strings can be created using three different methods in JavaScript:

  • Using Double Quotes ("): 
    let name = "John Doe";
  • Using Single Quotes ('): 
    let name = 'John Doe';
  • Using Template Literals (backticks `): 
    let name = `John Doe`;
2. String Length

You can determine the length of a string using the .length property:

let str = "Hello";
console.log(str.length); // 5
3. String Indexing

Each character in a string can be accessed by its index using the bracket notation. The index starts from 0:

let str = "Hello";
console.log(str[0]); // "H"
console.log(str[1]); // "e"
4. Escape Sequences

Special characters can be included in a string using escape sequences:

  • \n — Newline
  • \t — Tab
  • \\ — Backslash
  • \' — Single quote
  • \" — Double quote
let text = "He said, \"Hello!\"";
console.log(text); // He said, "Hello!"
5. String Methods

JavaScript provides a variety of built-in methods to manipulate strings:

  • charAt(index): Returns the character at a specified index.
    • let str = "Hello";
console.log(str.charAt(1)); // "e"
  • indexOf(substring): Returns the index of the first occurrence of a specified substring. If not found, it returns -1.
    • let str = "Hello";
console.log(str.indexOf("e")); // 1
console.log(str.indexOf("z")); // -1
  • substring(start, end): Returns a new string from the specified start index to the end index (not inclusive).
    • let str = "Hello";
console.log(str.substring(1, 4)); // "ell"
  • slice(start, end): Similar to substring, but allows negative indices (which count from the end of the string).
    • let str = "Hello";
console.log(str.slice(1, 4)); // "ell"
console.log(str.slice(-3));  // "llo"
  • toUpperCase(): Converts the entire string to uppercase.
    • let str = "hello";
console.log(str.toUpperCase()); // "HELLO"
  • toLowerCase(): Converts the entire string to lowercase.
    • let str = "HELLO";
console.log(str.toLowerCase()); // "hello"
  • trim(): Removes whitespace from both ends of the string.
    • let str = "  Hello  ";
console.log(str.trim()); // "Hello"
  • replace(searchValue, newValue): Replaces a substring with a new string.
    • let str = "Hello World";
console.log(str.replace("World", "JavaScript")); // "Hello JavaScript"
  • split(separator): Splits the string into an array of substrings based on a separator.
    • let str = "apple,banana,orange";
console.log(str.split(",")); // ["apple", "banana", "orange"]
6. Template Literals

Template literals, introduced in ES6, provide an easier and more powerful way to work with strings. They allow you to include variables inside strings using ${} for interpolation and support multiline strings.

  • String Interpolation: 
    let name = "John";
let greeting = `Hello, ${name}!`; 
console.log(greeting); // "Hello, John!"
  • Multiline Strings: 
    let multiLineStr = `This is
  a multiline
  string.`;
console.log(multiLineStr);
7. String Immutability

Strings in JavaScript are immutable. This means that once a string is created, it cannot be changed directly. Any method that modifies a string will return a new string.

let str = "Hello";
str[0] = "J"; // This will not work because strings are immutable.
console.log(str); // "Hello"

To modify a string, you would have to reassign it:

let str = "Hello";
str = "J" + str.slice(1);
console.log(str); // "Jello"
8. String Comparison

Strings can be compared using comparison operators (==, ===, !=, etc.) just like numbers. JavaScript compares strings lexicographically (character by character).

let str1 = "apple";
let str2 = "banana";
console.log(str1 < str2); // true because "apple" comes before "banana"
9. Unicode and Encoding

JavaScript strings are internally stored as UTF-16 encoded values. This means each character is represented by a 16-bit code unit. The charCodeAt() method returns the Unicode value of the character at the given index.

let str = "A";
console.log(str.charCodeAt(0)); // 65 (Unicode value for "A")
10. String Padding

JavaScript provides methods to pad a string:

  • padStart(targetLength, padString): Pads the current string from the start to a given length.
    • let str = "5";
console.log(str.padStart(3, "0")); // "005"
  • padEnd(targetLength, padString): Pads the current string from the end to a given length.
    • let str = "5";
console.log(str.padEnd(3, "0")); // "500"
